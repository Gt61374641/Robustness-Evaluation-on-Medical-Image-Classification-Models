## 2. 核心发现 H1:标准训练下复杂度-鲁棒性**非单调**(三数据集复现)

PGD full robust accuracy @ 0.1/255(PGD 的可区分区间在 sub-0.25/255):

| 模型 | 参数量 | chest | malaria | OCT |
|---|---|---|---|---|
| ResNet-18 | 11.7M | 0.30 | 0.90 | 0.56 |
| ResNet-34 | 21.8M | 0.20 | 0.83 | 0.46 |
| ResNet-50 | 25.6M | **0.04** | **0.72** | **0.33** |
| ResNet-101 | 44.5M | 0.08 | 0.72 | **0.27** |
| ResNet-152 | 60.2M | 0.26 | 0.91 | 0.60 |

**结论:**
- **U 形(非单调)**:复杂度阶梯**两端**(R18、R152)最鲁棒,**中间偏上**(R50/R101)最脆弱。
- **三个数据集一致** → 真信号,非单数据集偶然。
- **推翻范文的单调结论**("越简单越鲁棒"):最大的 R152 反而是最鲁棒之一。
- FGSM(单步)全程平缓、趋势相反 → 印证单步评估不可靠;PGD 才揭示真实脆弱性。

> 注:chest R50 在 ≥8/255 出现的"虚高"(如 0.62@16/255)是**大 ε 类塌缩伪影**(预测全压向多数类),由新增的 `pred_distribution.collapse` 诊断自动识别,已排除,不计入结论。

## 3. 核心发现 H2:对抗训练大幅提升鲁棒性,且复杂度有利

强评估 PGD-50 + 5 重启,clean / 全鲁棒 acc @ 8/255(**最终验证版**,2026-07-01):

| 模型 | chest | malaria | OCT |
|---|---|---|---|
| ResNet-50 | clean 0.80 / **0.53** ✅ | clean 0.96 / **0.90** ✅ | clean 0.74 / **0.64** ✅ |
| ResNet-152 | clean 0.82 / **0.65** ✅最强 | clean 0.96 / **0.91** ✅ | clean 0.25 / 0.03 ❌塌 |
| ResNet-18 | clean 0.62 / 0.00 ❌塌 | clean 0.46 / 0.00 ❌塌 | clean 0.41 / 0.16 ⚠️弱 |

**结论:**
- **标准模型 @8/255 真实鲁棒性 ≈ 0(强 PGD 下),AT 后 R50/R152 升到 0.5–0.9** → AT 巨幅提升,同协议对比铁证。
- **复杂度有利**:chest 与 malaria 上 R50/R152 都成功,且 **chest R152 最强(clean 0.82 / 0.65@8)**,clean 与 robust 均随复杂度上升 → 印证范文 AT 结论。
- **最小模型 R18 三数据集一致塌缩**(统一 warmup 协议下仍塌)+ **oct R152 塌缩**(最难 4 类 + 最深网络发散到平凡解)→ 作为诚实发现写入(已用 config 快照 + 逐 epoch 日志确认 warmup 已启用)。
- malaria 上 AT 最干净:R50/R152 在 8/255 仍有 0.90 鲁棒、clean 0.96。

## 4. 已知问题与处理(诚实呈现)

**问题:部分 AT 训练塌缩**(R18 各数据集、OCT 的 R152):clean 掉到随机水平,train loss 卡在 ln(类数)= 均匀输出。
- **原因**:预训练模型一上来被满预算(8/255)对抗样本打崩,早期发散到退化解;小模型 / 难任务更敏感。
- **修复**:已实现 **eps warmup + LR warmup**(前几个 epoch 线性升预算和学习率)+ OCT 加 epoch。
- **最终结论(2026-07-01 验证)**:warmup **救活了所有 R50 + chest R152**(如 OCT R50 从 clean 0.25 → **0.74 / 0.64@8**;chest R152 → **0.82 / 0.65@8**);但 **R18(三数据集)与 oct R152 加了 warmup 仍塌**(train loss 全程钉在 ln(类数))。已用运行时 config 快照确认 warmup 确实启用 → 这是**真发现**(最小模型普遍难 AT;最难任务上最深网络也发散),非配置疏漏。

## 5. 方法学贡献(相对范文的改进)

1. **可靠 PGD 评估**:强评估 PGD-50 + 5 重启;`pred_distribution` 诊断**自动识别大 ε 类塌缩伪影**(解决"大 ε 不能只裁图"的方法学问题)。
2. **标准/AT 训练配置统一**:自定义 PGD-AT 循环对齐标准训练(weighted CE + weight decay + cosine + AMP + 鲁棒-val 选点),保证差异只来自 AT。
3. **AT 稳定化**:eps/LR warmup 解决对抗训练塌缩。
4. **公平性**:同一固定测试集 + 同 ε 网格 + 双口径指标(full / conditional robust acc,解耦 clean 与鲁棒)。
5. **严谨划分**:分层 validation、OCT 训练子采样(确定性、跨模型共用)、malaria 患者级划分(0 泄漏)、结果 resume 防混(checkpoint mtime 校验)。
